# WetWorldWanderlust

Official studio website for Tanja & Christoph. The site presents the studio's human-led approach to strategy, digital products and UGC creation.

## Local preview

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production check

```bash
npm run build
```
